# 🌍 Multilingual Food Street Guide (Web + App)

## 📌 Project Overview
This project is a **C# ASP.NET Core Web API** for a **multilingual automatic narration system** designed for food streets.  
The system supports **multiple languages**, **GPS-based location detection**, and provides food information for both **web and mobile applications**.

👉 Project is developed as a **course assignment (đồ án)** and will be expanded in later phases.

---

## 🎯 Main Features
- 🍜 **Food information API** (name, description, etc.)
- 🌐 **Multilingual support** (Vietnamese, English – extensible)
- 📍 **GPS integration**
  - Detect user location
  - Suggest nearby food spots
- 🔊 **Automatic narration** (Text-to-Speech – future phase)
- 📱 Supports **Web & Mobile App** clients

---

## 🧱 System Architecture
- **Backend**: ASP.NET Core Web API (C#)
- **Frontend**: Web & Mobile App (planned)
- **Communication**: RESTful API (JSON)
- **Location**: GPS / Geolocation API
